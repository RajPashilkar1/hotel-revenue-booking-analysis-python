# 📊 Hotel Revenue & Booking Performance Analysis using Python

## 📌 Project Overview

This project focuses on analyzing hotel booking data to understand **revenue trends, booking patterns, occupancy rates, and customer behavior**. Using Python and data analysis techniques, the project converts raw hotel booking data into meaningful business insights that can help hotel management improve revenue and operational efficiency.

---

## 🎯 Business Objective

The objective of this project is to analyze hotel booking data and answer key business questions such as:

* Which **cities generate the highest revenue**?
* Which **booking platforms contribute the most bookings**?
* Which **room types are most profitable**?
* What is the **booking cancellation rate**?
* How do booking trends change over time?

The insights from this analysis can help hotels improve **pricing strategy, marketing decisions, and customer experience**.

---

## 📂 Dataset Information

The project uses multiple datasets structured in a **fact and dimension model**.

**Datasets included:**

* `dim_date.csv` – Contains date-related information
* `dim_hotels.csv` – Information about hotel properties
* `dim_rooms.csv` – Room type details
* `fact_aggregated_bookings.csv` – Aggregated booking metrics
* `fact_bookings.csv` – Detailed booking transactions
